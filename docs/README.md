## Quick Links

### Getting Started
1. Read [ARCHITECTURE.md](ARCHITECTURE.md) to understand the system
2. Follow [DEPLOYMENT-GUIDE.md](DEPLOYMENT-GUIDE.md) for step-by-step setup
3. Review [SECURITY.md](SECURITY.md) for security best practices

### Troubleshooting
- Check [TROUBLESHOOTING.md](TROUBLESHOOTING.md) for common issues
- Review logs with verbose mode: `bash -x ecr-artifact-manager.sh`
- Open a GitLab issue if problem persists

### Optimization
- See [COST-ANALYSIS.md](COST-ANALYSIS.md) for cost reduction strategies
- Review [SCALING-GUIDE.md](SCALING-GUIDE.md) for large-scale deployments
